Financial GUI Analysis Project

The goal of this project was to create a fully functioning Graphical User Interface (GUI) that would be able to import, process, visualise and analyse current and historical financial data. This GUI would be useful as a tool to aid old or new investors by helping them view trends and statistics easily and perform analysis that would enable them to develop a portfolio.

Importing the data

The starting point for this project was importing and processing the data from an online source. The data would be in four different forms:

  • Historical index values for the S&P 500
  • Current index values for the S&P 500
  • Historical stock data from companies in the S&P 500
  • Current stock data from companies in the S&P 500

The historical data would be the basis for the training of our Machine Learning classifier, and the current data would be used by this classifier to predict new labels. Yahoo Finance was used to import historical S&P 500 index values and current and historical stock data for companies in the S&P 500. A bot accessed the html code for the stock index and statistics for the individual companies and extracted the relevant information. These values were appended to a data frame which could be used for analysis. To collect data for the current S&P 500 index values I parsed data from Quandl, a source for financial datasets. The dataset was initialised using Pandas, and a series of issues were encountered when trying to extract and process the data including NaN value returns. These were solved individually using try and except commands, and a final data frame was saved to a csv (comma separated variable) file. The data frame had several rows for each company, each identifiable by the timestamp.


Machine Learning Analysis

From these datasets, a supervised machine learning algorithm was used to classify companies based on their current statistics. The labels attributed to each row was defined as a 1 if it was a buy, and a 0 if it was not. A buy was defined as a company that had a great increase in value across a year than the S&P 500 index. This ultimately meant that this company had beaten the market.


The features used to classify the data included the book/price ratio, enterprise value etc. These are basic metrics used by investors to calculate the quality of a company and assess whether a company is undervalued or overvalued relative to its stock price. Through using a machine learning algorithm, we are trying to automate the process of determining whether a stock is over or undervalued.


The labelled data was divided into training and test sets, with the training set to be used in the Machine Learning algorithm. The Support Vector Machine (SVM) algorithm was used to classify data, which uses a hyperplane defined by the closest data points (support vectors). This algorithm was able to return a list of companies that were predicted to beat the market by a pre-defined threshold given current financial data.


This algorithm is only is 54% accurate, therefore it is not a perfect tool for predicting the stock market. However, it is a useful tool for forming a basis for a portfolio that would require further research.



Graphical User Interface

Using the imported data and machine learning analysis software, a GUI was created that could encompass all this data into a package. Each section of the GUI is divided using tabs, which can be selected easily. The GUI allows the user to import the data from the sources specified previously with a loading bar. It allows the user to import data from a specific time period for the S&P 500. To expand upon this, I would like to incorporate other indexes such as the FTSE100 etc.


It allows you to visualise the imported data frame so you can assess some basic features. This includes the creation of a table of statistics for each company for a date selected by the user. Furthermore, it allows the user to plot the trend of the features for each company across the entire period, which can be selected using dropdown menus. These plots can be saved in the directory defined by the user.


The last section incorporates the machine learning analysis. The interface allows you to select the time period for investment and the threshold for deciding a buy. These parameters are used in the machine algorithm to generate a list of suitable companies, displayed in the right-hand side window, alongside the accuracy of the algorithm. Data from these companies can be visualised by selecting them from a dropdown menu, resulting in a plot being displayed below.



Conclusion

The GUI was successful at importing, processing, visualising and analysing the data set using machine learning, however there are several areas for improvement:

  • Importing data from other financial markets (FTSE 100).
  • Using other machine learning algorithms to improve the accuracy of the results.
  • Visualise the index performance against the stock performance, so that inferences can be made about stock performance.
  • Risk mitigation using graphical analysis such as the Simple Moving Average (SMA), Moving Average Convergence Divergence (MACD) and the Relative Strength Index (RSI).
