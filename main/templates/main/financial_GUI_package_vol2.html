Financial GUI Analysis Project, Vol 2

The goal for this project was to improve the previous GUI model to be able to import, process, visualise and analyse current and historical and financial data in more detail. This would mean employing more features, such as graphing tools and a stock screener. Furthermore, this updated version would have additional user interaction features to enable the tracking of stocks.

Importing the data

The GUI allows the user to import data from trusted sources on historical and current stock statistics. This is similar to the previous version of the software, however this updated version displays the files uploaded in the table in the top right corner, along with their size, source and link to the source.





Stock Screener

The stock screener feature allows users to append multiple conditions using drag bars to specify the maximum and minimum. After updating, it displays the list of companies that meet the conditions specified by the user. The ticker can then be selected and the features related to that company can be analysed on the right side of the page.



The right side of the page can be toggled between analysis of individual features, stock price, industry comparison and risk analysis. The individual features are grouped in individual tables; company basics, stock health, projected performance, company health, company past performance and stock ratios. These are colour coded based upon how they compare to other stocks, with red being in the bottom 20, yellow in the middle 60 and green in the top 20. The trend analysis tab plots the company’s stock price against the FTSE 100 index. The industry comparison tab compares the company’s key statistics with other companies in the same industry.









Portfolio Tracker

The portfolio tracker page enables users to append stocks to a portfolio, which are saved. These stocks can be analysed individually in a similar way as the stock screener. Analysis can be performed on features, trends and industry comparison.



Algorithmic Screener

The algorithmic screener performs the same function as the stock screener, however in this instance the conditions are already pre-defined. This has not been developed fully, and the only option available currently returns the top ten stocks based upon a cumulative ranked system for features. Again, these stocks can be analysed on the right. For further development I would hope to add additional screener options to provide greater depth in analysis for users.



Simulating Stocks

The simulation tab allows users to model the performance of potential investments, given the data stored in the database. It allows the user to select a stock and input an amount that they would like to invest. The software stores this data, and tracks changes daily based upon adjusted closing stock prices. The day-by-day history of investments can be seen on the right, along side plots that visualise the change in price of the stock. The overall analysis of all companies invested in can be seen at the bottom left, with features displayed such as profit and yield on investment.
