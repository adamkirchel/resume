Graphical Extraction Software Using MATLAB

During my University work I frequently encountered plots that were saved as image files. The data from these plots were inaccessible, unlike figures saved in MATLAB or Excel. The data in extracted from these plots would be helpful for analysing experimental data and assessing the accuracy between my own and otherâ€™s data.

I would achieve this by creating an application that could accept an image file, and automatically extract the datasets from it using an unsupervised machine learning algorithm. It would then output the data set as a saved csv file.

Cropping of the image

The datasets would be classified based on colour, and therefore the plots would have to be isolated so that any colours of external features or labels would be disregarded. The image was cropped so that only the area within the axes was considered. In some instances, there are labels within the axes that have the same colour as the data plots. These were cropped as well with the use of two user-defined points to create a rectangle.

Identification of axes

The axes origin were identified by binarizing the image matrix, classifying points as either black or not. The row and column indices for the longest length of consecutive ones was found, which identified origin. The end points of these lengths were found and defined as the end points of the axes.
The actual maximum and minimum x and y axes values were defined using a user-interface. Using these values, the axis were scaled and the image matrix was redefined with these values.

Using unsupervised machine learning techniques to separate different data sets

The data set was then separated using the K-Means ++ clustering algorithm. This separated the data into clusters defined by the user. The algorithm uses an iterative process to relocate cluster centres based on the mean value of the nearest datapoints so that they converge towards the cluster centres. The separated datapoints are then displayed on a figure, and the program asks whether the clusters created were expected.

Creating a GUI

This process was incorporated into a GUI so that an image can be selected from the directory. The plots are output onto the GUI for the user to make decisions within the process as specified above. Figure x displays the features of the GUI.
